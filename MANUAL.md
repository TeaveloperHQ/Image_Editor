## 1. 소개

- **이미지**: 크기 변경(픽셀·백분율), 자르기·회전·반전, 텍스트·그림 얹기.
- **PDF**: 여러 파일 결합, 페이지 분해·추출, 용량 줄이기·용지 크기 변경, 페이지 위에 텍스트·이미지·서명 얹기.
- **도장**: 이름·단체명으로 도장 이미지 만들기, 스캔한 직인의 배경 투명화.
- 편집은 **객체를 올려 마우스로 옮기고 크기를 맞춘 뒤 저장**하는 방식이라 직관적입니다.
- 설치가 필요 없습니다. `TeaveloperImageEditor.exe` 하나만 받아 더블클릭하면 실행됩니다.

## 2. 빠른 시작

- **이미지 크기 줄이기** → `이미지 크기 변경` 탭에서 파일 선택 → 픽셀/백분율 입력 → 저장.
- **PDF 합치기** → `PDF 결합` 탭에서 PDF 여러 개 추가 → 순서 정리 → `결합하여 저장`.
- **PDF 용량 줄이기**(서버 업로드용) → `PDF 크기/용량` 탭 → 파일 선택 → "용량 줄이기" 슬라이더 조절 → `용량 줄여 저장`.
- **사진에 글자·도장 넣기** → `이미지 편집` 탭에서 불러오기 → 도구 선택 → 미리보기 클릭으로 올리고 → 저장.
- **도장 만들기** → `도장 만들기` 탭에서 이름 입력 → `PNG로 저장` → 편집 탭의 "이미지/서명 선택"으로 불러와 찍기.

## 3. 화면 구성

위쪽 **탭**으로 기능을 이동하고, 맨 아래 **상태 표시줄**에 진행·완료·오류 메시지가 뜹니다.

- **이미지 크기 변경** — 이미지 한 장을 픽셀/백분율로 리사이즈
- **PDF 결합** — 여러 PDF를 한 파일로 합치기
- **PDF 분해·추출** — 페이지별 분리 / 원하는 페이지만 추출
- **PDF 크기/용량** — 용량 줄이기(압축) + 용지 크기 변경
- **이미지 편집** — 자르기·회전·텍스트/그림 올리기 (미리보기에서 직접 편집)
- **PDF 편집** — PDF 페이지 위에 텍스트·이미지·서명 올리기
- **도장 만들기** — 글자 도장 생성 / 스캔 직인 배경 투명화

---

## 4. 이미지 크기 변경 탭

1. `찾아보기…`로 이미지 파일을 선택합니다. (PNG·JPG·BMP·GIF·WebP·TIFF)
2. 변경 방식을 고릅니다.
   - **픽셀 지정** — 너비·높이를 입력. **비율 유지**를 켜면 한 박스 안에 비율대로 맞춰지고, 높이를 `0`으로 두면 너비에 맞춰 자동 계산됩니다.
   - **백분율** — 원본 대비 % (예: `50` = 절반).
3. `크기 변경 후 저장`을 누르고 저장 위치를 정하면 끝.

---

## 5. PDF 결합 탭

1. `PDF 추가…`로 합칠 PDF들을 불러옵니다. (여러 개 선택 가능)
2. 목록에서 순서를 정리합니다 — `위로` / `아래로` / `선택 삭제` / `전체 비우기`.
   - **위에서 아래 순서대로** 합쳐집니다.
3. `결합하여 저장`을 누릅니다. (2개 이상 필요)

---

## 6. PDF 분해·추출 탭

1. `찾아보기…`로 PDF를 선택하면 **총 페이지 수**가 표시됩니다.
2. 두 가지 방식 중 선택:
   - **① 페이지별로 분해** — 모든 페이지를 한 장씩 개별 PDF로 만들어 지정한 폴더에 저장.
   - **② 특정 페이지만 추출** — 페이지 범위를 입력해 한 PDF로 추출.
     - 형식: `1-3,5,8-10` (1~3쪽, 5쪽, 8~10쪽). 입력한 순서가 곧 결과 순서입니다.

---

## 7. PDF 크기/용량 탭

파일을 한 번만 선택하면 아래 두 작업을 모두 할 수 있습니다. 선택하면 **현재 용량**이 표시됩니다.

### 용량 줄이기 (서버 업로드용)
- **"조금 ↔ 많이"** 슬라이더 하나로 조절합니다. 오른쪽으로 갈수록 용량이 더 줄지만 사진 화질은 떨어집니다.
- `용량 줄여 저장`을 누르면 `원본 → 결과 (○○% 감소)`가 표시됩니다.
- PDF 안의 **이미지를 다운샘플·재압축**하는 방식이라, **스캔·사진이 많은 PDF**에 효과가 큽니다. 글자만 있는 PDF는 거의 줄지 않습니다(정상).

### 용지 크기 변경
- A4 / A3 / A5 / Letter / Legal 중에서 고른 뒤 `용지 크기 변경 후 저장`.
- **비율 유지(여백 추가)**를 켜면 내용 비율을 지키며 여백을 넣고, 끄면 용지에 꽉 채웁니다.

## 8. 이미지 편집 탭

미리보기는 **폭에 맞춰** 표시되고, **마우스 휠로 확대/축소**, **오른쪽 버튼 드래그로 화면 이동(팬)** 합니다.

1. `불러오기`로 이미지를 엽니다.
2. **회전 / 반전** — `↺ 90°` `↻ 90°` `좌우 반전` `상하 반전` 버튼으로 즉시 적용.
3. **작업 모드**를 고릅니다(처음엔 아무것도 선택 안 됨 — 실수 방지).
   - **자르기** — 미리보기에서 마우스로 영역을 드래그한 뒤 `선택 영역 자르기`. (그냥 클릭하면 선택 해제)
   - **텍스트** — 내용·폰트·크기·**색상(색상 칸을 눌러 팔레트에서 선택)** 을 정하고 미리보기를 클릭하면 그 자리에 글자가 **객체**로 올라갑니다.
   - **그림** — `이미지 선택…`으로 얹을 이미지를 고르고 미리보기를 클릭해 올립니다. (도장 PNG도 여기로 불러와 찍습니다)
4. 올라온 **객체는 드래그로 이동**, **파란 모서리 핸들로 크기 조절**합니다.
5. `저장…`을 누르면 객체가 그대로 **확정**되어 파일로 저장됩니다. (회전·자르기 시에도 자동 확정)

> 객체 선택 후 **Del** 키로 삭제, **Ctrl+Z**로 실행 취소.

## 9. PDF 편집 탭

PDF 페이지 위에 텍스트·이미지·서명을 **얹는** 편집입니다. (기존 PDF 글자를 고치는 것은 불가)

1. `불러오기`로 PDF를 엽니다. `◀ 이전` / `다음 ▶`으로 페이지를 넘깁니다.
2. **얹을 내용**을 고릅니다.
   - **텍스트** — 내용·폰트·크기·색상을 정하고 페이지를 클릭해 올립니다. `폰트 추가…`로 원하는 글꼴(.ttf/.otf)을 불러올 수 있습니다.
   - **이미지·서명** — `이미지 선택…`으로 서명 이미지나 도장 PNG를 고르고 페이지를 클릭해 올립니다.
3. 객체는 드래그 이동·모서리로 크기 조절. **페이지를 넘기거나 저장하면 그대로 확정**됩니다.
4. `저장…`으로 편집한 PDF를 내보냅니다.

> 미리보기 휠 확대/축소·우클릭 팬·**Del** 삭제·**Ctrl+Z** 실행 취소 모두 동일하게 동작합니다.
> 미리보기 글자는 화면 글꼴로 그려서 확정 후 실제 모양이 조금 다를 수 있습니다(위치·크기는 그대로).

## 10. 도장 만들기 탭

오른쪽에서 만들고, 가운데 미리보기로 확인합니다. (미리보기 배경의 teaveloper 엠블럼은 **투명 영역 확인용**)

### 글자로 만들기
1. 이름·단체명을 입력합니다. (한글/한자)
2. **모양** — `자동` / `원형` / `사각` 중 선택.
   - **자동**: 끝의 `인`(印)을 뺀 글자가 **5자 이상이면 사각**(단체), 그 이하는 **원형**.
3. **색상** — 붉은색 / 푸른색.
4. `도장 만들기` → 미리보기에 표시 → `PNG로 저장`.
   - 전서체 같은 도장 전용 글씨체를 쓰려면 `폰트 추가…`로 해당 폰트를 불러와 선택하세요.

### 스캔 투명화 (실제 직인용)
1. `스캔 이미지 선택…`으로 직인 스캔본을 고르면 **흰 배경이 자동으로 투명**해집니다.
2. 아래 값을 조절하면 **미리보기에 바로 반영**됩니다.
   - **배경 제거** — 값을 높일수록 옅은 부분까지 더 지웁니다. 잉크가 같이 지워지면 낮추세요.
   - **색 통일** — 켜면 번지거나 바랜 잉크를 선택한 색으로 균일하게 보정합니다. 끄면 원본 색 유지.
3. `PNG로 저장`.

> 저장한 도장 PNG는 **이미지 편집 / PDF 편집** 탭의 "이미지(서명) 선택"으로 불러와 원하는 위치에 찍을 수 있습니다.

## 11. 단축키와 공통 조작

| 동작 | 방법 |
| --- | --- |
| 확대 / 축소 | 미리보기 위에서 **마우스 휠** |
| 화면 이동(팬) | **오른쪽 버튼 드래그** |
| 객체 이동 | 객체를 **왼쪽 버튼 드래그** |
| 객체 크기 조절 | 객체의 **파란 모서리 핸들** 드래그 |
| 객체 삭제 | 객체 선택 후 **Del** |
| 실행 취소 | **Ctrl+Z** (이미지·PDF 편집) |
| 확정·저장 | **`저장…`** (미적용 객체는 저장 시 자동 확정) |

## 12. 자주 묻는 질문

**Q. PDF 용량을 줄였는데 거의 안 줄어요.**
글자만 있는 PDF는 줄일 이미지가 없어 변화가 적습니다(정상). 스캔·사진이 많은 PDF에서 효과가 큽니다.

**Q. PDF 안의 기존 글자를 고칠 수 있나요?**
아니요. 기존 텍스트 수정은 불가하고, 위에 **새 텍스트·이미지를 얹는** 방식만 됩니다. (이미지도 동일)

**Q. 도장에 전서체(직인 글씨체)를 쓰고 싶어요.**
무료로 재배포되는 한글 전서체가 거의 없어 기본 제공은 못 합니다. 보유하신 전서체 `.ttf`를 `폰트 추가…`로 넣어 선택하면 그대로 적용됩니다.

**Q. 편집 미리보기의 글자색이 실제와 약간 달라 보여요.**
미리보기는 화면 글꼴로 그리기 때문이며, 저장하면 선택한 글꼴·색으로 확정됩니다. 위치와 크기는 보이는 그대로입니다.

**Q. 설치가 필요한가요?**
아니요. `TeaveloperImageEditor.exe` 하나만 받아 더블클릭하면 됩니다. (.NET 설치 불필요)
